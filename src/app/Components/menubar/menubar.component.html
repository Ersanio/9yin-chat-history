About

This web application is able to display the Age of Wushu chat logs contained within this folder:

Snail Games USA\Age of Wushu\bin\[acc]\[file].xml

The selected chat log does NOT leave your browser, i.e. there is no server spying on your chatlogs.

You can verify this yourself by viewing the source code over at GitHub, or by opening your browser's developer tools and viewing the network traffic.

Made by Ersanio.
